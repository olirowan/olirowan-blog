I wanted to build a blog more than I actually wanted to write a blog.


The original aim was to find a project where I would be able to implement best security practices while getting some familiarity with full stack development. This required a platform that I could build where users could sign up and interact with something - like a simple blog site.


The problem was that I didn't have anything to write about. Eventually I thought it would be a good idea to get the site to a fairly stable state and publish it. Then I could continue it's development whilst documenting the changes along the way - providing the content for blog posts.




Built With

This site is developed using Flask.

A micro web application framework written in Python.

The design is created using the Bootstrap framework.

The icons used throughout the site have been obtained from freepik and Font Awesome.

The back-end relies on the use of SQLAlchemy to allow for interaction between the application and the database. The SSL has been obtained for free from the Lets Encrypt certificate authority.


A lot of resources have carried the development of this site. Notably - the Flask tutorials written by Miguel Grinberg and Charles Leifer. Also, the /r/webdev community, and the posts on the Security Stack Exchange site have been a great source of information.
