About

I started making this site because I wanted to familiarise myself with full stack development. I wanted to create a web application so I could learn how to build an API, how to securely store user information, how to handle credentials and also so I could have a blog to document it.



Built With

This site is developed using the Flask micro web application framework, written in Python. The design is created using the Bootstrap framework to manage the CSS. The back-end relies on the use of SQLAlchemy to allow for interaction between the application and the database. The SSL has been obtained for free from the Lets Encrypt certificate authority.


A lot of resources have carried the development of this site. Notably - the Flask tutorials written by Miguel Grinberg and Charles Leifer. Also, the /r/webdev community, and the posts on the Security Stack Exchange site have been a great source of information.





Built Because

This site is being created with two aims in mind.

The first is to have a personal blog, with the experience of building it along the way - rather than signing up to a blogging platform. This provides an insight into full stack development along with full control of the application.


The second is to learn and implement best security practices across the site, from forcing HTTPS on all pages to handling user interaction securely. This means providing features that require user interaction such as the /notes aspect in order to learn how to manage and store credentials, sanitise user input, offer multi factor authentication, and minimise site vulnerabilities.
